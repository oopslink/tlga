# 小学霸冒险记 (The Little Genius Adventurer)

AI 驱动的学习积分管理系统，通过 Claude Code + 纯 Markdown 实现游戏化学习跟踪。

## 项目简介

小学霸冒险记是一个面向小学生的学习激励系统，将日常学习任务转化为冒险游戏。通过金币、经验值(XP)、星星三种货币体系，配合等级晋升、连击奖励、隐藏成就等机制，让学习变得有趣。

## 工作方式

本项目不包含任何代码，完全基于 **Claude Code + Markdown** 运行：

1. 玩家每天向 Claude Code 汇报学习完成情况
2. Claude 根据 `CLAUDE.md` 中的规则自动计算积分
3. 所有数据以 Markdown 文件形式存储在本仓库中

## 使用方法

### 每日汇报

在项目目录下打开 Claude Code，直接告诉它今天完成了什么：

```
今天完成了：学校作业、奥数简单版正确率90%、抛接球50个、背诵了一篇课文
```

Claude 会自动：
- 创建当日日志文件
- 计算金币/XP/星星奖励
- 检测连击和隐藏成就
- 更新玩家总进度

### 每周结算

周末时告诉 Claude：

```
这周结算一下吧
```

Claude 会检查周挑战榜、副本完成情况、发放周奖励。

### 兑换奖励

```
我想用金币兑换 / 我想用星星抽许愿池
```

### 查看进度

```
看看我现在的进度 / 我离升级还差多少
```

## 游戏系统概览

- **主线任务**: 学校作业、奥数、抛接球、语言学习
- **奥数加成**: 普通/火焰/星光/钻石/王者五种模式，最高 ×3 倍金币
- **连击奖励**: 连续 3/7/14 天完成主线，获额外奖励
- **等级系统**: Lv1 小学徒 → Lv10+ 知识大师，各级解锁不同特权
- **隐藏成就**: 8 个隐藏成就等待解锁
- **许愿池**: 用星星抽取愿望
- **大目标翻倍**: 完成长期目标，金币/星星可翻倍

详细规则见 `rules/game-rules.md`。
